# 2026-05-22

## READY STOCK DTS policy

- Root V1 READY STOCK bulk flow must apply shop-level DTS per region:
  - `TW=1`
  - `SG=2`
  - `TH=2`
  - `MY=2`
  - `PH=2`
  - `BR=3`
- `update_global_dts` remains `days_to_ship=1` as the global catalog baseline.
- Operator-visible correctness depends on the per-shop `update_shop_item_dts` calls, not on Shopee clamp behavior.

## PRE ORDER DTS policy

- Root V1 Shopee modal now exposes editable `PRE ORDER DTS by Region` inputs.
- `products.shopee_days_to_ship.pre_order` is the persisted source of truth for the root app.
- The Shopee modal sync button pushes region-level live updates through `update_shop_item_dts` with `is_pre_order=true`.
- Default PRE ORDER DTS remains `10` for all operating regions unless the product overrides it.
