🌐 Global → 매장 sync 정책 일괄 설정

    

      SG/TW/TH/MY/PH/BR 6개 활성 매장의 Global Product sync 정책을 아래 선택한 flag 로 일괄 설정합니다.
      Shopee API 에 get_sync_field 가 없어 현재 정책을 읽을 수 없으므로, 실행 시 모든 flag 를 명시적으로 set 합니다.
      상품명/옵션명 ON 시 매장 내 해당 항목이 Global 영문명으로 통일됩니다 (매장별 커스터마이즈 덮어씀).
